# Required Reading:
	Review Unix shell Command.

•	change directories: cd directory

•	list files in directory: ls directory

•	move/rename file: mv oldfile newfile

•	copy a file: cp oldfile newfile 

•	delete a file: rm file 

•	remove an empty directory: rmdir directory

•	create an empty directory: mkdir directory 

•	read a file using an editor: nano ,vim…..

•	find file: find location (e.g /) file

•	compression: tar cvzf somedir.tar.gz somedir

•	decompression: tar xvzf somedir.tar.gz

•	Add users: adduser --home path of home Directory username

•	Add group: groupadd newgroup

•	Change file owner: chown newowner filename

•	Change group: chgrp newgroup filename

•	Change permission modes: chmod xxxx somefile.sh

•	modify many details of a preexisting user account:  usermod
