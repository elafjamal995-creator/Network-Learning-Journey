# CONFIGURING DHCP, DNS & WEB SERVERS ###

1. Draw necessary topology, decorate and comment
2. Configure IP to the router interface and static IP addresses to the servers
3. On the DHCP Server device, create DHCP pools, subnet mask, no of useable IP, default gateway and dns address.
4. Go to every PC and change option to DHCP.
5. Configure DNS server to translate web server IP to gtech.com
6. Go to web server and modify index.html
8. Test accessing the web server from any PC by entering gtech.com on the browser url.

* CONTINUATION --- EMAIL + FTP SERVERS 

1. Finish configuring static IP address to Email and FTP servers
2. Go to FTP server and create username and password for logins.
3. Test from any PC if FTP service is accessible from the CMD
4. Go to Email server and create domain name, usernames and passwords for logins
5. Go to the PCs and login emails there
6. Test ssending emails from one PC to another.
