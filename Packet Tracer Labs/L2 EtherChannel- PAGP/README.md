# PAgP ETHERCHANNEL CONFIGS #######

1. Draw necessary topology, decorate and comment
2. Identify trunk links, and configure PAgP channel group number and mode.
Note: both sides should have the same channel group number,
mode can only be desirable/auto or desirable/desirable but not auto/auto
3. Enter the port channel number created and configure it as trunk interface.
4. Display etherchannel information using the command 'sh etherchannel'
