REMOTE ACCESS USING TELNET CONFIGS

1. Draw necessary topology, decorate and comment
2. Assign IP addresses to the router and host devices
3. Configure management IP address and default gateway on the switch.
4. Configure hostname, domain name, enable password, username and password.
5. Configure local login and transport input telnet on vty interfaces.
6. On the Admin-PC ping the management IP address of the switch --- should be successful
7. On the Admin-PC, telnet to the switch using command telnet 192.168.2 ....
