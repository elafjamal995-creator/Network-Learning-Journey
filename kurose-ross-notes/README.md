# Computer Networking: A Top-Down Approach (8th Edition)

This folder contains my study resources and practical work for the textbook **"Computer Networking: A Top-Down Approach" (8th Edition)** by Jim Kurose and Keith Ross.

## 🎯 Core Objectives

1. **Chapter Summaries:** Documenting the most important concepts, protocols, and technical notes for **Chapters 1 through 6**.
2. **Wireshark Labs:** Completing and documenting the official Wireshark network analysis labs found at the end of each of these chapters.
